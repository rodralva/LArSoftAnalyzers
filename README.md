# LArSoftAnalyzer
Analyzer that dumps:
- G4 stage  SimPhotons arriving a optical channels (Truth)
- Raw OpDetWaveforms (+ channel and timestamp)
- OpHits from Hit Finder on OpDetWaveforms
- Added MC Truth info (x,y,z, deposited energy....)
